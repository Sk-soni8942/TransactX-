# 💰 TransactX Backend

A robust and scalable backend system for managing financial transactions, users, and role-based access control (RBAC). Built using Node.js, Express, and MongoDB.

---

## 🚀 Features

- 🔐 User Authentication (JWT-based)
- 👥 Role-Based Access Control (Admin, Analyst, Viewer)
- 💰 Transaction Management (Income & Expense)
- 🔍 Advanced Filtering (category, type, date range)
- 🛡️ Secure API with middleware protection
- 🍪 Cookie-based authentication (access & refresh tokens)

---

## 🧠 Tech Stack

- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
